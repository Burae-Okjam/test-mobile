# 에겐/테토 유형 테스트 모바일 웹사이트

간단한 설문을 통해 에겐/테토 유형을 파악하는 모바일 대응 웹사이트의 기본 뼈대입니다.

## 사용 방법

```bash
python -m http.server
# 또는 다른 정적 서버로 index.html 실행
```

## 구조

- `index.html` – 진입 페이지
- `src/` – 스크립트 모듈 (API 연동, UI 렌더링 등)
- `styles/` – 기본 스타일

## 테스트

현재 자동 테스트는 없습니다. `npm test` 명령은 자리 표시자 메시지만 출력합니다.
